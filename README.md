# 🎲 Jogo dos 3 Dados em Java

Projeto simples desenvolvido em **Java** utilizando **Swing** para criar uma interface gráfica.

## 📌 Descrição

Este projeto simula um jogo de dados entre o jogador e o computador.

Cada lado rola **3 dados**, e o vencedor é aquele que tiver a maior soma dos valores.

O jogo possui uma interface gráfica simples com botão para rolar os dados e exibir o resultado.

## 🚀 Tecnologias utilizadas

- Java
- Java Swing (interface gráfica)

## 🎮 Como funciona

1. O jogador clica no botão **Rolar Dados**
2. O sistema gera números aleatórios entre **1 e 6**
3. O programa soma os valores dos dados
4. O sistema compara jogador vs computador
5. O resultado é exibido na tela

## 📷 Exemplo

Jogador: 12  
Computador: 9  

Resultado: **Você venceu!**
